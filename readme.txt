# doubleguid plugin
# Plugin for B3 (www.bigbrotherbot.com)
# www.ptitbigorneau.fr

doubleguid plugin (v1.2) for B3

Installation:

1. Place the doubleguid.py in your ../b3/extplugins and the 
doubleguid.ini in your ../b3/extplugins/conf folders.

2. Open your B3.xml file (default in b3/conf) and add the next line in the
<plugins> section of the file:

<plugin name="doubleguid" config="@b3/extplugins/conf/doubleguid.ini"/>
